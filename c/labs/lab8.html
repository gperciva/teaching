Create a program that lets you move around on a map.  When you
move onto special characters, a special event should happen.





You arrive at the park.

+------+
|.+.~~~|
|.@...~|
|.....~|
|.._=_.|
+------+





In this example, the @ is the player (as before).  If
the player moves onto a . then nothing happens.  A
~ is water, so it should print out a message like
"your feet get wet".  The underscore _ is a park
bench, so a message like "you sit down" might be appropriate.  The
= is a table, so the message might be "you dance on
the table.  Other people stop and stare at you".  The
+ is a tree, so maybe "you sit against the tree and
fall asleep" could work.  The | and -
are boundaries of the area; if the player steps there, the program
should exit.
