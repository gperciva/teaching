Lab assessment

Grades

Lab material will be graded by a lab demonstrator in class. There are 5 marks for each exercise, with a total of 8 graded exercises. Since labs are worth 40% of your final grade, this means that every point in an exercise is worth 1% of your final grade.

The criteria is:

  • 2 marks: code meets specifications.

    (the bullet points under "your tasks")

  • 1 mark: good code style.

    (the style for this class is specified in good style)

  • 2 marks: two questions about your code.

    (we will change your code, then ask how your program would behave differently -- will the printed output change? will it still compile? will the program produce unpredictable output?)

Rationale

More than 50% (some people estimate 80-90%!) of a programmer's job is reading code, not writing it. Maintaining a large code base is a difficult task even if the code was written well; if it was written poorly, it becomes almost impossible. In addition, the easier your code is to read, the fewer mistakes you will make in the first place!
