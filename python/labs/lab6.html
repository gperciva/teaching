Exercise 15: Complex Arithmetic; Frequency Response

Background

In the last lab, we constructed oscillators by placing poles on the unit circle in the z-plane. This concept can be extended by providing the objects we build with an input, and implementing them with poles (and zeros) inside the unit circle.

To calculate the frequency response of such a system, we can follow the procedure described in the "Subtractive Synthesis" chapter of mcm.pdf. This describes an easily-rememered graphical method to calculate the magnitude and phase of a system's transfer function given the position of the poles and zeros. In summary:

  1. Take a position on the unit-circle corresponding to the frequency of the input signal.
    2. Construct vectors from each pole to that position.
  2. Calculate the vector product of these vectors, producing the result P
  3. Repeat the procedure for for the zeros yeilding Z. Remember that to make a causal system, you probably ended up with zeros at the origin.
  4. The frequency response is the result of the vector division of Z by P

Maths homework

To be able to calculate frequency responses and design filters easily, it is important to be fluent at vector arithmetic, so the the prep for this section contains a bit of practice at that. You should be able to do these sums "by hand" (which, these days, means using a calulator which can only do real numbers).

Work out the following sums, expressing the results in polar form:

  1. (4∠π/3)(3∠π/2)
  2. (-2∠π/3)³
  3. (2+j3)(1.5∠-3π/2)
  4. j(1+j)
  5. (3∠-π/4)(-2∠-3π/2)
  6. (3∠π/2)÷(4∠π/3)

If a system is running at a sample rate of 44100Hz, has poles at z = 0.995∠±π/10 and zeros at z = 0.9∠±π/10, at what frequency does it have maximum gain? And what is that gain?
